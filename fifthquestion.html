Favicon(Favorite Icon):-

Favicon is just like a dp of a webpage.Favicon sets the image on the top of the webpage. It is a tiny icon which displays on the browers address bar.Favicon is written in the head part of the html file.Favicon works is to just put the image in the website top with the address bar so that it improves the user experience and website looks maintainable.The icon link will be put in href section in the head tag.
